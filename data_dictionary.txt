#### housing_tax.csv
PID: Uniquely identifies each parcel (use as final index)
data type: int64
variable type: quantitative

TtlVal_AsrYr: Assessment Value by the Ames City Assessor
dtype: float64
variable type: quantitative

GLA: Gross Living Area
dtype: float64
variable type: quantitative

total_returns: Total Number Tax Returns Filed (proxy for households)
dtype: float64
variable type: quantitative

adj_gross_inc: Adjusted Gross Income (AGI)
dtype: float64
variable type: quantitative

agi_per_ret: Adjusted Gross Income per Return, higher the AGI per Return for a given zip code, higher the wealth
dtype: float64
variable type: quantitative

perc_business_ret: Ratio of total business income divided by AGI, higher the ratio more businesses operate in that zip code
dtype: float64
variable type: quantitative

perc_farm_ret: Percentage of returns filed by farmers, higher the percentage for a given zip code higher the rural population
dtype: float64
variable type: quantitative

perc_umemp_ret: Ratio of returns that claimed unemployment to total returns, higher the ratio worse are the economic conditions
dtype: float64
variable type: quantitative

perc_ssn_benefits: Ratio of returns that claimed social security benefits to total returns, higher the ratio lower would be the wealth for that zip code
dtype: float64
variable type: quantitative

perc_student_loans: Ratio of returns that claimed student loans to total returns, higher the ratio younger the population in that zip code
dtype: float64
variable type: quantitative

perc_child_credits: Ratio of total child credits to AGI, higher the ratio more children live in that zip code
dtype: float64
variable type: quantitative

Perc_earned_inc_tax: Ratio of total earned income credits divided by AGI, higher the ratio higher the population of poor households in that zip code
dtype: float64
variable type: quantitative
